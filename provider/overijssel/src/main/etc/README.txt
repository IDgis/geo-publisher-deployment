GIS Publicatieomgeving Provider
-------------------------------

Installatie:
- Uitvoeren install-service.bat
- Aanpassen provider.conf:
	- wachtwoorden
	- paden
	- hostnamen
- Uitvoeren 'services.msc'
- Aanpassen login voor service 'GIS Publicatieomgeving Provider'
- Uitvoeren start-service.bat

Verwijderen:
- Uitvoeren stop-service.bat
- Uitvoeren delete-service.bat
